To Report an Error or Omission

Please email errors@detroitledger.org with any relevant details, including a link to the page and a description of the problem. Any corrections will be noted here and on the relevant pages.

Data Sources

Data on this site comes from the State of Michigan and is freely available online. Many datasets are from the Center for Educational Performance and Information. Testing data is from the Michigan Office of Educational Assessment and Accountability.

Acknowledgements

The Ann Arbor District Library generously hosts the Detroit Ledger's Michigan School Data project. The AADL does not provide editorial guidance and is not responsible for the content or accuracy of this site.

Code

The static HTML files for this site and a simple search application are available free of charge under the CC0 license. The HTML pages were generated by a simple Django app using data stored in MongoDB in the style of the Chicago Tribune's schools app. The parsers for the State data can be provided but are currently undocumented context specific. Please contact Matt Hampel for a copy and more detailed explanation of our process.
